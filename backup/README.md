# 旧版页面备份

从站点根目录移入的历史首页，避免与新版 `index.html` 混用。

| 文件 | 说明 |
|------|------|
| `index3.html` | 原「旧版入口」页面（先天八卦布局） |
| `index2.html` | 另一版中间态首页 |

## 说明

- 主站已移除「旧版入口」链接，线上仅使用根目录 `index.html`。
- 资源路径已改为 `../static/...`，便于从本目录本地打开时加载样式与脚本。
- `yi.js` 内通过 `document.write` 引入 `./static/js/data.js`，从 `backup/` 打开时 data 路径可能仍不正确；完整功能请将文件移回站点根目录再访问。

## 恢复方式

将需要的 html 移回仓库根目录，并把其中的 `../static` 改回 `./static`（以及链接中的 `../`）。
